# 🎮 Jogos e Projetos

> Coletânea de jogos e projetos front-end para praticar e aprimorar o desenvolvimento web.

Repositório com diversos miniprojetos construídos em **HTML**, **CSS** e **JavaScript**, voltados para estudo, prática e exploração de novas técnicas de desenvolvimento.

## ▶️ Como executar localmente

```bash
# Clone o repositório
git clone https://github.com/michellycruz/jogos-e-projetos.git

# Acesse a pasta do projeto desejado
cd jogos-e-projetos/jogo-da-velha

# Abra o index.html no navegador
```

Cada projeto é independente — basta abrir o `index.html` da pasta correspondente.
